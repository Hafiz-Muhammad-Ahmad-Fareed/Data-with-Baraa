What is Programming Language?
=> A Language which computers can understand.

What is a Program?
=> A set of instructions written in a programming language to solve a particular problem.

Types of Languages:
=> Natural Language (Urdu, Hindi, English)
=> High-Level Language (Python, JavaScript)
=> Low-Level Language (C, C++)
=> Machine Language (Binary)

How Python Works?
=> source_code(.py) -> Interpreter [Compiler -> Byte Code + Libraries -> VM -> Machine Code] -> Output
